# GMIT — 굿모닝인테리어 홈페이지 (초안)

서울 노원구 상계동 인테리어 업체 **굿모닝인테리어**의 원페이지 홈페이지 초안입니다.
피드백 수집용으로 GitHub Pages에 배포합니다.

## 구성
- **진입 게이트** — `시공사례 먼저보기` / `견적요청 먼저하기` 두 갈래
- **시공사례** — 호프집·부대찌개집·치킨집 Before/After + 욕실·주방·부분시공 (사진은 분위기 참고용 스톡)
- **견적요청** — 5단계 위저드(공간·용도 → 공사 범위 → 시기·예산 → 레퍼런스 → 연락처)

## 로컬에서 보기
```
python -m http.server 8000
# http://localhost:8000
```

## 참고
- `index.html` 한 파일로 완결됩니다(이미지는 base64로 내장).
- 견적 폼과 사진 첨부는 아직 프론트엔드 데모입니다. 실제 접수를 받으려면 백엔드 연동이 필요합니다.
- 시공사례 사진은 Pexels·Unsplash 무료 라이선스 스톡이며, 실제 시공 사진으로 교체할 자리입니다.
